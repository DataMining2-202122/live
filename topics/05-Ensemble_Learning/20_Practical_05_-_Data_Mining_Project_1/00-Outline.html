Estimating Sales Level

The dataset (train, test) contains details of a set of products sold in a number of stores. Your task is to predict the sales level for products within stores.

Dataset

The dataset is relatively simple, so you can probably guess the purpose of the attributes form their names. But just in case, we have attributes

  • Product_Code - Unique product code
  • Product_Weight - Weight of product
  • Product_Fat_Content - Product fat content
  • Product_Salt_Content - Product salt content
  • Product_Visibility - The % of total display area of all products in a store allocated to this product
  • Product_Type - The category to which the product belongs.
  • Product_MRP - Maximum Retail Price (list price) of the product
  • Store_Code - Unique store code
  • Store_Opened - Year store opened for business
  • Store_Size - Size of the store in terms of ground area covered
  • Store_Location_Type - Type of city in which the store is located
  • Store_Type - Type of store.

and target

  • Product_Store_Sales Sales level of this product at this store.

Machine Learning Task

The data is relatively simple but needs some cleaning and preparation. So expect sections:

  • Data Load and Preparation
  • Pre-EDA EDA and Data cleaning
  • EDA on train dataset
  • Feature Engineering/Selection
  • Model selection/training/optimisation

Grading

The grading breakdown for this assignment is given below:

  • Data Load and Preparation: 0%
    • This is a trivial task so won't insult you by giving marks for this.
  • Pre-EDA EDA and Data cleaning: 20%
    • Describe the meaning and type of data (scale, values, etc.) for each attribute in the data file.
    • Verify data quality: Are there missing values? Duplicate data? Outliers? Are those mistakes? How do you deal with these problems?
  • EDA on train dataset: 20%
    • Explore relationships between attributes and target.
    • Explore relationships between attributes: Look at the attributes via scatter plots, correlation, cross-tabulation, group-wise averages, etc. as appropriate.
  • Feature Engineering/Selection: 15%
  • Are there other features that could be added to the data or created from existing features?
  • Model selection/training/optimisation: 25%
  • Create multiple regression models. Adjust parameters as appropriate.

  • Assess how well each model performs (use training/test data and/or cross validation as appropriate).

  • Model Performance: 20%

    • How well does model perform on hidden dataset? Grade will be based on ranked RMS.

Notes:

  1. The _Pre_EDA__ and the EDA components will be graded together as the distinction between both activities is movable.
  2. Describe your thought process / rationale with brief (markdown) comments.
