Review of Pandas - German Credit

In this practical we are going to analyse a small dataset. The purpose of this practical is to ensure you are happy with both basic pandas operations and data mining process (EDA) that were covered in Data Mining 1 module.

The aims of this lab session are:

  • Install / Verify a Suitable Python Environment
    You probably have a working python installation from last semester, but if not we will use a Python 3.9 distribution and all additional modules that will be used during this course have been tested to ensure no incompatibilities at this time.

  • Review of Basic Data Exploration Skills.
    We will take a small dataset based on the German Credit dataset.

  • Develop Familiarity with Moodle, in Particular Moodle Activities (Quiz)
    To help direct your data exploration, mentioned above, we have a small Moodle quiz based on the server tips. This quiz is formative (i.e., not summative) so does not contribute towards your overall grade.
