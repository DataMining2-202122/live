Pre-Processing Data

Now that you have some general understanding of the dataset that you are working with, we can start to think about modelling.

Train/Test split

Since we want to have an unbiased measurement of our model quality we perform the usual train/test spilt, with two slight modifications.

  • use stratified split (on target) to ensure train and test sub-datasets have approximately the same percentage of samples of each target class as the complete set --- important for
  • use a fixed seed for the random number generator to allow comparisons between students (i.e. grading (for later practicals)).

from sklearn.model_selection import train_test_split
df_all = df

df_train, df_test = train_test_split(df_all, stratify=df_all.Churn, test_size=.25, random_state=42)

Data Preparation

Then after the train/test split you revisit the EDA and general procedure here is

  1. Attribute selection.

    • Are there any variables that can be eliminated?
    • Select one or two subsets of attributes with good predicting capability and motivate your choices (based on your EDA).

  2. Handle missing values.

  3. Eliminate outliers.

  4. Discretisation of numeric attributes.

    • Do not forget to indicate which discretisation method did you use, if discretisation is used.

  5. Normalisation.

    • Motivate the need of normalisation (if needed at all).
