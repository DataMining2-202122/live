Loading Data

The dataset for this practical consists of two CSV files:

  • churn
    Records on 3333 customers, each customer is described by 20 attributes, and a binary target indicating whether the customer churned.
  • states
    contains information on each of the states (Full name and coordinates).

A description of the meaning of each attribute in churn is:

  1. State: categorical variable that indicates the state where the customer lives.
  2. Account Length: integer variable that indicates how long the account has been active.
  3. Area Code.
  4. Phone: Phone number.
  5. Int'l Plan: binary variable indicating whether the customer has an international plan.
  6. VMail Plan: binary variable indicating whether the customer has a voice mail plan.
  7. VMail Mesgs: integer variable that indicates the number of voice mail messages.
  8. Day Min: continuous variable that indicates number of minutes the customer used the service during day time.
  9. Day Calls: integer variable that indicates the number of calls during day time.
  10. Day Charge: continuous variable that indicates how much was charged for using the service during day time.
  11. Eve Min: continuous variable that indicates number of minutes the customer used the service during evening time.
  12. Eve Calls: integer variable that indicates the number of calls during evening time.
  13. Eve Charge: continuous variable that indicates how much was charged for using the service during evening time.
  14. Night Mins: continuous variable that indicates number of minutes the customer used the service during night time.
  15. Night Calls: integer variable that indicates the number of calls during night time.
  16. Night Charge: continuous variable that indicates how much was charged for using the service during night time.
  17. Intl Min: continuous variable that indicates number of minutes the customer used the service to make international calls.
  18. Intl Calls: integer variable that indicates the number of international calls.
  19. Intl Charge: continuous variable that indicates how much was charged for international calls.
  20. CustServ Calls: integer variable that indicates the number of calls to customer support service.
  21. Churn?: binary class variable.

Chrun

  • Standardise column names
    As you can see from above, the column names need to be changed to simplify access later so:

    • remove any quotes (or any other punctation chars) in names.
    • replace spaces by underscore.
    • convert to title case (using title case will mean your features will not clash with any existing python methods)

    In other words change the column names so that you can use the dot notation to access columns in pandas.

  • Also I would do a very quick EDA (using pandas_profiling or dtale)

  • Drop any problematic columns.

  • Save cleaned files to data subfolder.
