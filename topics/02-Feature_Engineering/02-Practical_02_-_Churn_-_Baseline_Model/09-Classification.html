Classification

  • How well do the DecisionTree, LogisticRegression, RandomForest, ... classifiers (by default) work?

  • How well do the variaous metrics behave in terms of being a reilable indicator or performace?

  • To simplify the use of ROC with cross validation I have written a python script which you can use as follows:

1
2
3
4
5
from sklearn.linear_model import LogisticRegression
classifier = LogisticRegression(solver="lbfgs")

import my_classifier_metrics
ax, cm, metrics = my_classifier_metrics.cvClassifier(X_train, y_train, classifier)
  • The function my_classifier_metrics.cvClassifier outputs the various metrics
1

3
4
Accuracy: 85.87%
Precision: 53.28%
Recall:   20.17%
F1 score:   29.26%

and returns the ROC plot, the average confusion matrix, and a dictionary of metrics. The ROC plot should look like
